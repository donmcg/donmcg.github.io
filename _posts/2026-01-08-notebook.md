* Of immediate interest is to create small schematic diagram snippets.
I'd prefer to use svg for this, rather than png images from KiCad because
I want small file sizes and tidy presentations.

It seems that inkscape may be the tool to make this happen.

---
<br>

The first trick I've learned in inkscape is  

{% highlight inkscape %}
Path > Object to Path
Path > Combine
Path > Union
File > Document Properties > Display > Resize to content:
File > Save As > foo.svg 
{% endhighlight %}

This will render an svg file that can be loaded using my *include figure.html*
script.
