I've added some content to the SU piston project but now want to add
some images.

Back in this [post]({% post_url 2025-10-27-notebook %}) I worked out
how to remove metadata.  Now I also want to reduce the file size of
the images....

Then in this [post]({% post_url 2025-12-13-notebook %}) I worked out
a procedure. But it isn't complete and doesn't produce small 
png files needed for github.io.  

This is an updated procedure to do that...

---

<br>

-0. Install programs
{% highlight bash %}
   $ brew install pngquant oxipng
{% endhighlight %}

-1. Use Preview to crop the image and save it in PNG format to a local file:  
*dirty_my_image.png*

-2. Optional look at the file to see the metadata
{% highlight bash %}
m2 $ exiftool dirty_my_image.png
{% endhighlight %}

-3. Compress the file
{% highlight bash %}
   $ pngquant --quality 20-50 dirty_my_image.png
 # this creates the file dirty_my_image-fs8.png
{% endhighlight %}

-4. Remove the metadata
{% highlight bash %}
   $ oxipng -o 4 -i 0 --strip safe dirty_my_image-fs8.png
{% endhighlight %}

-5. Double check the metadata
{% highlight bash %}
   $ exiftool dirty_my_image-fs8.png
{% endhighlight %}

-6. Clean file is ready for publication now
{% highlight bash %}
   $ mv dirty_my_image-fs8.png clean_my_image.png
{% endhighlight %}

<br>

To facilitate the above process here's a bash function "compPNG":

{% highlight bash %}
function compPNG {
  # compactify PNG file to minimize file size on a website
  # usage: compPng pngFilename
	
  # pngFilename="./path/to/file.png"
  pngFilename=$1
 
  if test -f $pngFilename; then
    echo "Compactifying the file: $pngFilename"
  else
    echo "File does not exist: $pngFilename"
		return 1
  fi

  # case insensitive check for .png extension
  # Enable case-insensitive matching
  shopt -s nocasematch
  if [[ "$pngFilename" == *.png ]]; then
    echo "The file, $pngFilename, has a .png extension."
  else
    echo "The file, $pngFilename, does not have a .png extension."
		return 1
  fi

  pngquantOutfile="${pngFilename%.*}-fs8.${pngFilename##*.}"
  echo "pngquantOutfile: "$pngquantOutfile

  comp_pngFilename="${pngFilename%.*}-comp.${pngFilename##*.}"
  echo "comp_pngFilename: "$comp_pngFilename

  # destroy existing pngquantOuftile
  if [ -f $pngquantOutfile ]; then rm $pngquantOutfile ; fi

  # compress the file
  # pngquant will normally insert the -fs8 substring
  # here it is inserted explicity
  pngquant --quality 20-50 --output $pngquantOutfile $pngFilename

  # rename the pngquant file
  mv $pngquantOutfile $comp_pngFilename

  # remove the metadata from the file
  # oxipng will not change the comp_pngFilename 
  oxipng -o 4 -i 0 --strip safe $comp_pngFilename

};
{% endhighlight %}


<br>
... And if those .png files are the only ones in the directory:

{% highlight bash %}
for f in `ls -1 *.png`; do echo $f; compPNG $f; done
{% endhighlight %}
