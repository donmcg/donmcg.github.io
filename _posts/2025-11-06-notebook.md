For what I want, it seems that I need to be able to create my own classes
for scss.  With that, I should able to style my elements with the
**{: .myclass }** syntax.

And to do that I need to
- create local *_sass* directory at root of the site
- add a custom styles file in the *_sass* directory: *_fairhillbillies.scss*
- import the custom scss file into my main stylesheet
    - create a new *assets/main.scss* file (overriding the default one which is located at /Users/donmc/.gem/ruby/3.4.1/gems/minima-2.5.1/assets/main.scss)
    - within the new *main.scss* import "minima" and also the new styles:

{% highlight scss %}

---
# Only the main Sass file needs front matter (the dashes are enough)
---

@import "minima";
@import "fairhillbillies";
{% endhighlight %}

Initial contents of *_fairhillbillies.scss*:

{% highlight scss %}
    // _sass/_fairhillbillies.scss
    .my-custom-class {
      color: blue;
      font-size: $small-font-size;
    }

    .another-class {
      background-color: lightgray;
      padding: 10px;
      @include relative-font-size(1.9375);
    }
{% endhighlight %}

---

Now, in this post's markdown file I have the paragraphs

{% highlight liquid %}
This uses class my-custom-class
{: .my-custom-class }

This uses class another-class
{: .another-class }
{% endhighlight %}

Which renders as:

This uses class my-custom-class
{: .my-custom-class }

This uses class another-class
{: .another-class }

Awesome! And notice that *$small-font-size* and *relative-font-size()*
are honored. These come from minima.
